## Run the App
- git clone <this repo>
- npm install
 - npm start
 ## Change states of Buttons
 - modify purchasingChannelCode
 - Add2Cart Visible (0 || 1)
 - PickUp in Store Visible (0 || 2)
 - purchasingChannelCode is located in the products.json file (modified the name)
